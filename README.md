# Adventura

Minimal static site built with plain HTML, CSS and JavaScript.

## Quick start

Open the site:

- Double-click `index.html` in the project root, or
- Run a simple static server:

```bash
# Node
npx http-server . -c-1


## Project structure
```

/
├─ index.html
├─ css/
├─ js/
├─ assets/ # images, fonts, etc.
└─ README.md

```

## Deployment
- GitHub Pages: push to `gh-pages` or use the repository settings to serve from `main`/`docs`.
- Netlify/Vercel: drag & drop `index.html` or connect repo (no build step).

## Notes
No build step required. Replace placeholders and add more scripts only if you introduce a bundler/tooling.
```
